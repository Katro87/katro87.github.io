## Setup for GitHub Pages

1. Create a repository named `katro87.github.io`
2. Upload all files maintaining the folder structure
3. Enable GitHub Pages in repository settings
4. Your site will be live at `https://katro87.github.io`

## Customization

1. Update contact information in all HTML files
2. Modify color scheme in `assets/css/style.css` (CSS variables section)
3. Add your own project images and descriptions
4. Update social media links
